### Progress Percentage
```

nextTier = PLATINUM (3000 pts required)
userPoints = 1850

range = nextTier - currentTier = 3000 - 1500 = 1500
progress = userPoints - currentTier = 1850 - 1500 = 350
percentage = (progress / range) Ã— 100 = (350 / 1500) Ã— 100 = 23%
```
