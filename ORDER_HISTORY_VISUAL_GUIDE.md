## Color Scheme

- **Header Background**: Coffee Brown (#6F4E37)
- **Card Background**: Latte Foam (#FFFAF0)
- **Primary Text**: Dark Coffee (#3E2723)
- **Secondary Text**: Coffee Brown with opacity
- **Accent**: Coffee Brown for prices and buttons
- **Status Badges**:
  - Completed: Green background with darker green text
  - In Progress: Orange background with darker orange text
  - Cancelled: Red background with darker red text

## Interactive Elements

1. **Search Bar**: 
   - Real-time filtering
   - Icon indicator
   - Clear button (when text present)

2. **Filter Chips**:
   - Active state: Filled with primary color
   - Inactive state: Outlined/surface color
   - Bold text when selected

3. **Order Cards**:
   - Tap/Click "View Details" to expand
   - Shows truncated items when collapsed
   - Shows full details when expanded
   - Smooth transition animation

4. **Status Badge**:
   - Pill-shaped design
   - Color-coded for quick recognition
   - Bold, small text

## Responsive Behavior

- **Mobile Portrait**: Single column, full-width cards
- **Tablet/Landscape**: Maintains single column for consistency
- **Scrolling**: Smooth vertical scroll with proper padding
- **Keyboard**: Search bar properly handles keyboard input
